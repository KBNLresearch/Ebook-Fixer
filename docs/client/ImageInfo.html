ImageInfo

ImageInfo(element, section, asset, replacementUrl)

This class is used to store information about images in the book. A list of these is generated when the e-book is opened.

Constructor

# new ImageInfo(element, section, asset, replacementUrl)

Constructor for the ImageInfo object, this is to organize the code such that it's clear what information is being used where and how

Name Type Description

Element of the image from the pre-rendered (not final) document

Path & type of the image, {href: path to image inside the book, type: type of the image, ...}
