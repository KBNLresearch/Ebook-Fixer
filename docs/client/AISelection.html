# <AISelection setStage currAiSelected setCurrAiSelected setAiAnnotationList setSentence currImage ebookId imageId />

The AISelection component handles selection of various AI types, such as Google Vision or Microsoft Azure. After this step the AIAnnotator component will deal with generating the actual annotations using that AI.
