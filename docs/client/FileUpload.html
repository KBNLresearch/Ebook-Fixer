# <FileUpload setEbookFile setEbookId setEbookTitle />

The FileUpload handles uploading the epub and sending it to the server. It supports both drag and drop and choosing a file with a system window. It checks the file type to be an epub.

PropTypes:
Name Type Required Description Default

setEbookId func Yes Function to set epub id once received response from server
setEbookTitle func Yes Function to set epub title

# inner handleSubmit(e)

This Function checks that a file has been submitted / dropped and is called automatically when the user submits the form (by clicking the upload button) It checks the file type of the submission (by calling a helper function) And sends the file to the server using the API.
